## Getting started

To install this Node.js application on your local device, follow these steps:

```bash
git clone https://github.com/KODI-Kommunen-Digital/Kodi.Kusel_BE.git
cd Kodi.Kusel_BE
npm install
```

To start the program, run this command:

```bash
npm start
```

The API will start running on your localhost.
